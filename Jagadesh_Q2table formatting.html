Formatting Tags
Syntax Tag Name Function Example
<b> Bold This tag is used to make text in bold.<strong> is recomended instead of this tag. Bold
<i> Italic This tag is used to make the text in italic format.<em> is recomended. Italic
<u> UnderLine This tag is used to Underline the text UnderLine
<del> Delete This tag is used to indicate the deleted text Delete
<ins> Insert This tag is used to indicate the inserted text Insert
<sub&> Subscript This tag is used for subscripting the text

This is Subscript
<sup> Superscript This tag is used for superscripting the text

This is Superscript
<small> small This tag is used to display the text in small size compared to default text small
<big> Big This tag is used to display the text in larger size compared to default text Big
<mark> Mark This tag is used to highlight the text Highlight
